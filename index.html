01 — 品牌定位
Kwosk 是谁，
为谁而做。
品牌的骨架。所有决策——内容、定价、合作——都从这里出发。
核心定位句
"穿得好，不需要人看出来。"
品牌在说什么
自信不需要外部认证。质感是给自己的，不是表演给别人看的。
目标客户心态
已经有自己审美，不需要被潮流牵着走的城市男性。
产品定位
值得留下来的东西。300GSM，穿了会知道差别在哪里。
品牌个性
内敛、有立场、不解释、不讨好。
为什么这个定位长远有利润
身份
卖的是认同感，
不是产品功能
溢价
身份认同品牌
支撑更高定价
忠诚
认同你价值观的客户
不比价，不等促销
三个探索过的方向
方向 C — 反潮流基本款主义 ✓ 选定
"你不需要排队证明你有品味。" — 最贴近核心定位，骨气最强。
方向 B — 城市上班族的场外衣橱
"下班后，才是你自己。" — 用作内容和 campaign 场景叙事，不作主定位。
方向 A — 本地慢时尚（已排除）
"少买，买好。" — 需要更宽的产品线支撑，现阶段不适合。
02 — 品牌语言
怎么说话，
比说什么更重要。
每一句 caption、每一条 DM、每一个标题，都在建立品牌印象。
说话方式
应该有的感觉
简短有力 不解释、不卖力 有立场 冷静的自信
永远不要出现
感叹号 "超级""最好" 促销语气 过度解释功能
文案对比示例
✓ KWOSK 的方式

"300GSM。你会感觉到差别。"
"不是每个人都需要被看见。"
"很快了。"
✗ 不要这样写
"超舒适面料！穿上就爱上！"
"最高质量的棉花！"
"展示你的个性！让大家注意你！"
"快来购买！！"

04 — Lookbook 拍摄
安静、真实、
不刻意。
镜头里的人知道自己是谁，不需要表演。
参考词：quiet confidence · urban realism · film still
三个核心场景
1
精品咖啡馆
城市男性最自然的"自己时间"场景
拍什么
坐着看手机 / 端咖啡杯
望向窗外的侧脸
手放桌上的细节特写
光线
窗边自然光优先
早上9–11点最好
不要正午硬光
2
城市建筑 / 停车场
工业感背景衬出衣服质感
拍什么
靠墙站立、不看镜头

手插裤袋、低头看地
光线
阴天散射光最稳定
黄昏前1小时也好
避开中午直射
3
家居 / 室内极简空间
展示"私下的质感"，最贴近品牌定位
拍什么
坐在椅子上看书
整理衣领的细节
只拍下半身 + 地板
背景
白墙 / 米色墙优先
背景越干净越好
避开杂乱家具
镜头角度
腰部以上 ¾ 侧身
最主力角度。展示版型和肩线。

05 — 博主策略
找对的人穿，
不是找多的人。
5个精准的博主，胜过50个不相关的。核心逻辑：免费送 + 专属 code = 可追踪的销售渠道。
核心公式
免费送衣
成本：产品成本
+
专属 Code
每人独立追踪
=
数据筛选
谁真正带货
博主筛选标准
找这种博主
穿搭低调有质感
内容有真实生活感
不靠大牌 logo 撑场面
文字有自己的想法
粉丝 3k–15k，互动真实
避开这种博主
专拍大牌 flex 内容
每帖都是广告合作
风格太 hype / streetwear
互动都是 emoji，无真实留言
地区不在 KL / 大城市
DM 话术
第一次 DM — 参考框架
"Hey [名字]，我们是 Kwosk，马来西亚本地男装品牌，主打 300GSM 重磅棉 T-shirt，给那些对细节有要求的人做的。

看了你的穿搭，觉得风格很搭，想问你有没有兴趣收一件试穿？没有任何发帖要求，只是希望如果你穿了觉得好，愿意分享就好。

会附上一个你专属的 code，你的 follower 用了会有折扣。"
关键：说"没有强迫发帖"——反而让真心喜欢的人更愿意发。
Promo Code 设置
格式KWOSK-[博主名缩写][折扣]
例子KWOSK-ADAM10 / KWOSK-RAFI15
折扣建议RM10–15，不超过15%
追踪方式Shopee 后台折扣码使用记录
复盘时机发布后 21 天，对比每个 code 数据
寄货包裹内容
T-shirt — 折好放好，不要皱
手写感卡片 — 品牌故事一句话，让人感觉是专门为他准备的
拍摄灵感参考 — 2–3 张你希望他们呈现的 mood（不是强制，是给灵感）
专属 promo code 卡片 — 清楚写明 follower 使用方式
寄出去的不只是 T-shirt，是你的品牌第一印象。包裹的质感直接影响博主愿不愿意拍。
