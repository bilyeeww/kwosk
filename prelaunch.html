01 — Shopify 设置
你的店铺是
顾客下单的地方。
买家从博主或 Instagram 点进来，第一眼看到的就是你的 Shopify。专业度直接影响他们愿不愿意掏钱。
基本设置 必须完成
✓
自定义域名
用 kwosk.com 或 kwosk.my — 不要用默认的 myshopify.com。域名一年约 RM50，是最基本的专业度信号。
必须
✓
品牌视觉对齐
Shopify 主题颜色、字体、Logo 要跟 Brand Playbook 一致。Ink Black 底，Warm White 文字，Kwosk Red 作点缀。
必须
✓
产品页面完整度
每件 T-shirt 要有：多角度图片（最少5张）、详细尺码表、面料说明（300GSM，100% Cotton）、洗涤说明、预计发货时间。
必须
✓
尺码对照表
马来西亚买家最怕买错尺码。做一张清楚的尺码表：S/M/L/XL 对应实际胸围、肩宽、衣长（cm）。放在每个产品页面。
必须
✓
付款方式设置
开通 FPX（本地网银）+ 信用卡。可以用 Shopify Payments 或接 Billplz / iPay88。FPX 是马来西亚买家最常用的付款方式。
必须
✓
运费设置
设定西马固定运费（建议 RM8–10）。可以考虑满 RM150 或 RM200 免运费，提高客单价。东马运费另设。
必须
转化率提升 建议完成
✓
限量库存显示
在产品页面显示"只剩 X 件"。Shopify 可以设置当库存低于某数量时自动显示。制造真实的稀缺感。
建议
✓
退换货政策页面
写清楚退换条件（例如：7天内未穿着可退换，附上单据）。有明确政策的店铺，买家下单信心更高。
建议
✓
WhatsApp / 联络方式
放一个 WhatsApp 联络按钮或 email。马来西亚买家在付款前很可能想先问问题，有地方问 = 下单信心 +30%。
建议
✓
Discount Code 功能
在 Shopify 后台建好每个博主的专属 code，测试确认可以使用。发布前全部 code 要测试过一遍。
建议
最重要一步：发布前自己用手机完整走一次购物流程，从点进产品页 → 加入购物车 → 结账 → 付款。任何卡顿都要在发布前修好。
02 — 建立信任感
顾客不认识你，
凭什么给你钱？
新品牌最大的障碍是信任。不是价格，不是设计——是"这个品牌靠不靠谱"。以下是解决这个问题的方法。
信任信号清单
📦
开箱内容提前拍
发布前拍一支 Reel：从收到包裹、开箱、摸面料、穿上身的完整过程。让买家提前"体验"收货感觉，消除不确定感。这是转化率最高的内容类型之一。
📏
真人试穿不同尺码
拍不同体型的人穿同一款 T-shirt（不同尺码），让买家对号入座。你是 160cm，可以展示 oversize fit。如果能找一个 175cm 的朋友一起拍对比更好。
🧵
面料细节特写视频
用手拉扯面料展示弹性和厚度。与普通 tee 做重量对比（放上秤）。300GSM 的卖点要让人"看到"和"感受到"，不能只靠文字说明。
✍️
创始人故事
发布前1–2周，发一个 Story 或帖子讲你为什么做 Kwosk。不用长，3–4句就够。人们买品牌之前，先买创始人的故事。你的设计背景和对质感的坚持，本身就是说服力。
💬
提前收集反馈
发布前把样品给3–5个朋友穿，请他们拍一张穿着照和写一句真实感想。发布日用这些真实反馈作为 social proof。不需要正式评价，朋友穿着的截图或引言就够。
🔒
Shopify SSL + 安全标志
确保你的 Shopify 域名有 https（SSL 证书）。在结账页面显示安全付款图标。马来西亚买家对网购安全意识越来越高，这些细节会影响他们敢不敢输入信用卡资料。
Instagram 账号本身的可信度
账号要有起码 9–12 帖内容
发布日前，账号不能看起来刚开始。至少要有内容累积。
Highlight 要设置好
至少建 3 个 Highlights：产品、尺码参考、品牌故事。新访客第一件事就看这里。
Bio 要有 Shopify link
发布日把 bio link 换成 Shopify 产品页，不是首页。减少一个点击步骤。
03 — 内容就位
发布前，内容
要全部拍好。
发布期间你没有时间拍新内容。所有素材要在 Week 1 开始前全部准备好，发布时只需要点发布。
必备内容素材 发布前完成
✓
主 Campaign Reel（已有）
你已拍好的 campaign video — 这是发布日的主炮。确保剪辑、字幕、音乐版权都就位。
必须
✓
4款设计的产品图（Shopify 用）
每款至少5张：正面、背面、侧面、面料特写、穿着效果。白底或品牌色底，干净清楚。这是买家决定下单的最直接视觉。
必须
✓
Lookbook 照片（Instagram 帖用）
按照 Lookbook 拍摄指南，最少12张可用的静态照片，覆盖3周所有帖子用量。按照场景和角度分类存好。
必须
✓
面料特写短片（15秒）
手摸面料、拉扯弹性、与薄 tee 重量对比。用于 Story 和 Reel。这是最能转化"犹豫中"的买家的内容。
必须
✓
Teaser 图（黑底白字）
Day 1 发布用。简单的黑底白字静态图，配一句话。用 Canva 或你自己的设计工具做，10分钟可以搞定。
必须
✓
全系列4款并排图
4款设计放在一张图里（Carousel 封面或单图）。发布日主帖和 Day 12 揭晓用。要有 Kwosk 品牌一致的视觉风格。
必须
✓
博主 brief 文件
准备一份简单的 PDF 或图片：品牌定位一句话、希望传达的感觉、2–3张 mood 参考图。放进寄货包裹，也可以 DM 发给博主。
建议
内容批量拍摄建议：找一个周末，一次过拍完所有内容。准备好两套穿搭，去3个场景（咖啡馆 + 户外 + 室内），拍够3小时。比每次拍一点效率高很多，而且视觉风格更一致。
04 — 广告投放
RM100–300
怎么用最有效。
预算有限，必须精准。不要撒网，要用刀。现阶段博主 seeding 的效益比付费广告高，广告是辅助，不是主力。
预算分配建议

RM 50–100
RESERVE
备用 / 加码效果好的帖
哪个帖子反应最好，把剩余预算全压在那里
RM 50
Instagram Boost 怎么设置才有效
1
受众设置
不要用"自动受众"。手动设置：马来西亚、男性、18–35岁、兴趣标签选 menswear / streetwear / lifestyle / coffee。
2
目标选"访问网站"
不要选"更多帖子互动"（会得到很多 like 但不会去买）。选"访问网站"直接导流到你的 Shopify 产品页。
3
用最好的内容去 boost
不要 boost 普通帖子。等发布后看哪个帖子的自然互动最好，把预算押在那里。一般等 6–12 小时看数据再决定。
4
Reel 优先于静态帖
Instagram 算法现在大幅偏向 Reel。同样预算，boost Reel 的触达人数比静态帖高 2–3 倍。
现阶段不建议做的广告
✗
Facebook 广告（Meta Ads Manager）
设置复杂，学习曲线高，RM300 以下预算效益低。等品牌有一定数据积累再考虑。
✗
Google 搜索广告
没有人在 Google 搜索你的品牌名，搜索广告现阶段完全没有意义。
✗
付费博主发帖（付现金）
现阶段用免费送货换内容，效益比付钱高。付费合作等第一波数据出来，找到真正带货的博主后再谈。
广告的本质：RM100–300 的预算，博主 seeding 带来的真实口碑，效益远高于付费广告。广告是用来放大已经有效的内容，不是用来替代有机增长。先让内容自然跑，有效果了再加钱。
05 — 运营后勤
发布那天，
后台要零故障。
顾客下单了，后面发生什么决定他会不会再买。把这些准备好，发布日才能专心在内容上。
库存 + 包装 发布前必须就位
✓
确认库存数量
4款设计各备多少件？在 Shopify 设置真实库存数字。建议首批每款 20–30 件，不要过多积压资金，也不要太少无法制造稀缺感。
必须
✓
包装材料就位
纸袋或盒子、防水袋（雨季必备）、品牌贴纸或 hang tag、手写感感谢卡。包装是"开箱体验"的一部分，也是买家会不会拍照分享的关键。
必须
✓
快递合作确认
确认用哪家快递（J&T / Poslaju / Shopee Xpress）。了解取件时间和费用。Shopify 可以接 EasyParcel 一键比价出单，省很多时间。
必须
✓
发货时间承诺
在 Shopify 产品页面清楚写明：下单后几个工作天发货。建议承诺 2–3 个工作天，实际尽量在 1 天内发出。超出承诺 = 差评，提前发出 = 惊喜。
必须
顾客体验 建议完成
✓
自动确认邮件设置
Shopify 自动发送的订单确认邮件，把内容改成 Kwosk 的语气和品牌风格。不要用默认模板，这是顾客收到的第一封邮件。
建议
✓
包裹里放感谢卡
简单一张卡，写上定位句 + 一句真诚的感谢。可以加上"拍照 tag 我们得到 RM15 下次购物折扣"——鼓励 UGC 内容。
建议
✓
WhatsApp 快速回复模板
准备3–5个常见问题的标准回答（尺码、发货时间、退换货）。发布日询问量会暴增，有模板才不会手忙脚乱。
建议
发布日 Checklist（Day 15 当天）
09:00 Instagram 主帖发出，Shopify 产品页确认上架
09:30 确认博主已发帖，转发到你的 Story
10:00 XHS 同步发布，中文文案
全天 回复所有 DM、留言、WhatsApp 询问，在1小时内回复
晚上 Instagram boost 开始投放，Story 更新今日已售数量
发布日最重要的工作不是内容——是回复。每一个问问题的人都是潜在买家。发布日第一时间回复所有询问，转化率可以提高50%以上。把这一天空出来，不要安排其他事情。
